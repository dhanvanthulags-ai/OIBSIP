# Explore X - Travel Agency Landing Page

## Project Description
A modern, responsive landing page for a travel agency called "Explore X". The page showcases travel packages, company benefits, and encourages users to book their next adventure.

## Features Implemented
✅ Sticky navigation bar with smooth scrolling  
✅ Hero section with compelling headline and CTA button  
✅ 3 feature cards highlighting company strengths  
✅ About section with company statistics  
✅ Responsive footer with contact info and social links  
✅ Mobile-friendly design (tested on 480px, 768px, 1200px)  
✅ Consistent color palette (Professional Blue theme)  
✅ Clean typography with readable font sizes  
✅ No overlapping elements with intentional spacing  

## Color Palette
- **Primary:** #2C3E50 (Dark Blue-Gray)
- **Secondary:** #3498DB (Bright Blue)
- **Accent:** #E74C3C (Coral Red)
- **Background:** #ECF0F1 (Light Gray)

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- FontAwesome Icons (CDN)

## Responsive Design
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: Below 768px

## How to Use
1. Open `index.html` in any web browser
2. Navigate using the menu links
3. Click "Book Your Adventure" button
4. Resize browser to test responsiveness
