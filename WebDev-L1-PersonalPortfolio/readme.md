# Dhanvanth U - Personal Portfolio

## Project Description
A modern, dark-themed personal portfolio website showcasing my web development skills, projects, and experience. Built with a professional design featuring cyan and purple neon accents.

## Features Implemented
✅ Sticky navigation bar with smooth scrolling  
✅ Hero section with profile photo and intro  
✅ About me section with detailed background  
✅ Skills section with 6 skill cards  
✅ Projects section with 3 project cards  
✅ Contact section with multiple contact methods  
✅ Social media links (GitHub, LinkedIn, Twitter)  
✅ Download CV button  
✅ Responsive design (mobile-friendly)  
✅ Smooth animations and hover effects  
✅ Neon glow effects and modern styling  

## Color Scheme
- **Primary Dark:** #0f0f1e
- **Secondary Dark:** #1a1a2e
- **Accent Cyan:** #00d4ff
- **Accent Purple:** #7c3aed

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Smooth scroll, Active nav)
- FontAwesome Icons

## Responsive Design
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: Below 768px

## How to Use
1. Open `index.html` in any web browser
2. Navigate using the menu links
3. View projects and skills
4. Contact information available in the Contact section
